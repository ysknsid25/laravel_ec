# Laravel FREE E-Commerce Software

Features Provided
- Products
- Cart
- Checkout
- Categories
- Customers
- Orders
- Payment
- Couriers
- Employees

# Simplified DOCKER setup
### In your teminal, issue these commands

- RUN `docker-compose up -d --build`
- RUN `cd project && make install`
- OPEN [http://localhost:8000](http://localhost:8000)
